About Breadcrumbs

This is our breadcrumbs application for CS658. We are currently working on development and attempting to improve functionality whenever we can. In this area, we will explain the use of the app and our plans moving forward.

Application Use
  1. On applicaton start up, the user will be presented with our logo and a welcome menu.
  2. The user will then be able to navigate to a sidemenu and will be presented with three options.
    • Welcome: Which will always navigate the user back to this welcome page.
    • Start Run: Which will direct them to the page to start a run.
    • Account: Which will allow a user to view their last run info, should there be any.
  3. Should the user select "Start Run," then the users location and time will begin being recorded and they will be presented an application that records their time while moving.
  4. From here the user can choose to end the run. Ending the run will display the final results then allow the user to return to the "Start Run" page to begin again.
Planned Development
We plan to continue to develop and refine this application as we move forward and work to improve its functionality and services.
  • We plan to separate the timing system from the location system so that we are then able to implement a pause run button.
  • It is also our goal to work on implementing better data collection and account systems, eventually implementing route call back and a "dynamic splits system that will allow a user to see specific info between any two points on a run."
  • We would also like to implement a VR functionality to track the routes you had taken prior for a sort of reverse navigation. (Ever hear of Hansle and Gretle?)
Thank you for using our application!
